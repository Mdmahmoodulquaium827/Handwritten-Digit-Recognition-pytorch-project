# Handwritten Digit Recognition using PyTorch

This project uses a Convolutional Neural Network (CNN) to recognize handwritten digits from the MNIST dataset. It includes:

Training the model on MNIST training data

Testing and evaluating accuracy on MNIST test data

Saving the trained model for later use

Predicting digits from new images

The project demonstrates how to preprocess images, train a deep learning model, and use it for real-time digit predictions.
